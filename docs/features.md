# 功能模块文档

本文档详细描述了 Super Forum System 的各个功能模块。

---

## 用户认证系统

### JWT 认证机制
- Access Token + Refresh Token 双令牌机制
- Token 自动刷新和轮换
- Token 黑名单机制（登出后立即失效）

### 用户管理
- 用户注册
- 用户登录
- 用户登出
- 密码加密存储（bcrypt）

---

## 社区管理

### 社区创建与管理
- 创建新社区
- 社区详情页面
- 社区列表展示
- 社区推荐（根据帖子内容智能推荐）

### 成员角色系统
- **member** - 普通成员
- **moderator** - 社区管理员
- **owner** - 社区拥有者

### 自动加入机制
- 创建社区时，创建者自动成为社区拥有者（owner）
- 在某个社区中发帖时，如果用户还不是该社区成员，会自动加入后再发帖

### 成员管理功能
- 加入/退出社区
- 封禁/解封用户
- 管理员晋升/降级
- 社区所有权转让

### 导航流程
- 首页动态（Home Feed）：显示所有可见帖子，支持分页
- 热门内容：显示全局热门帖子、热门社区、活跃用户
- 我的社区：显示用户已加入的社区列表
- 社区详情页：显示社区信息和社区内帖子
- 发帖页面：可预选社区或从社区详情页直接发帖

## 热度排名

### Reddit 经典热度算法
```
hot_score = log10(|score|) + (timestamp / 45000)
```

- `score = upvotes - downvotes`
- `45000` = 12.5 小时（时间衰减常数）
- 新帖和高赞帖都能获得合理排名

### 功能特性
- 时间衰减机制
- 分数权重计算
- Redis 实时热度排行
- 全局热门和社区热门
- 侧边栏热门内容（热门帖子、热门社区、活跃用户）

### 上传限制
| 文件类型 | 最大大小 |
|---------|---------|
| 图片 | 10MB |
| 视频 | 100MB |
| 普通文件 | 50MB |

## 超级管理员系统

### 管理员控制面板（Dashboard）
- 系统统计数据（用户数、帖子数、评论数、社区数）
- 操作类型统计图表
- 操作趋势图表

### 用户管理
- 用户列表（支持搜索、筛选）
- 用户详情查看
- 冻结/解冻用户

### 帖子管理
- 帖子列表（支持搜索、包含已删除）
- 帖子详情查看
- 软删除/恢复帖子
- 硬删除帖子（永久删除）

### 评论管理
- 评论列表（支持搜索、包含已删除）
- 软删除/恢复评论
- 硬删除评论（永久删除）

### 审计日志
- 操作日志记录（所有管理员操作）
- 日志查询（支持按管理员、操作类型、目标类型筛选）
- 操作统计和趋势分析

## 全文搜索

### 技术实现
- PostgreSQL 全文搜索 + GIN 倒排索引
- 中文分词支持（zhcfg 配置）
- 相关性排序（ts_rank）
- 搜索关键词高亮显示（ts_headline）

### 搜索范围
- 帖子搜索
- 评论搜索
- 用户搜索
- 混合搜索（帖子+评论+用户）

### 搜索功能
- 搜索建议（快速预览）
- 分页支持
